# Archived Legacy Code

This directory contains legacy code that has been superseded by the current architecture.
Files are preserved here for reference only.

## Contents

- `Controller/Main.py` — Original monolithic controller with inline training logic (PyQt5-based)
- `Controller/GUI.py` — Original PyQt5 GUI
- `Controller/TUI.py` — Text-based UI (legacy)
- `Controller/m0_devices.py` — Legacy M0 device handler (replaced by M0Device.py)
- `Controller/requirements.txt` — Old pip requirements (replaced by pyproject.toml + uv)
- `main.py` — Original entry point
- `src/` — Older peripheral classes used by the legacy main.py
- `z_obs_arc/` — Obsolete archived files
